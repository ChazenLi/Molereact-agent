# Tools

This folder contains stateless analysis and utility tools used by the agent.

Key modules:
- `analysis.py`: Molecule analysis and property reporting.
- `advanced_analysis.py`: Extended analysis and scoring helpers.
- `inventory.py`: Stock check integration for precursor availability.
- `fg_cycle_detector.py`: Functional group and cycle detection.
- `planning.py`: Planning utilities for candidate ranking.
- `visualization.py`: Per-stage visualization helper.
- `base.py`: Tool registry and base tool interfaces.
